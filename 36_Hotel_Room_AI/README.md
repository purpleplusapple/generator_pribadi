# Hotel Room AI

A luxury hotel interior design application powered by AI.

## Features

- **Design Wizard:** Guided flow to create hotel room makeovers.
- **AI Transformation:** Uses Replicate API to redesign rooms.
- **Editorial Styles:** Choose from curated luxury styles (Boutique, Modern, Classic).
- **History:** Save and compare your designs.

## Structure

- `lib/screens/home`: Dashboard layout.
- `lib/screens/wizard`: 5-step vertical wizard flow.
- `lib/theme`: "Boutique Linen" design system.
- `lib/widgets`: Reusable components (SwipeCompare, StaggeredGrid).

## Setup

1.  Clone repository.
2.  Run `flutter pub get`.
3.  Run `flutter run`.
