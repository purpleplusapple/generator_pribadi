# Study Class AI

Premium Academic & Study Room Design Assistant powered by AI.

## Features
- **Editorial Dashboard**: Explore study styles and focus zones.
- **Wizard Rail**: Step-by-step guidance to redesign your study space.
- **Moodboard Styles**: 28+ curated styles (Dark Academia, Modern Minimal, etc.).
- **Smart Controls**: Customize lighting, desk size, and storage.
- **Premium Results**: High-quality AI generations tailored for study environments.

## Architecture
- **Framework**: Flutter
- **State Management**: Provider / ValueNotifier
- **Backend**: Replicate (Nano Banana) + RevenueCat
