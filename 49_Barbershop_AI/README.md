# Barbershop AI

**Premium Barbershop Interior Design Application**

Design your dream barbershop with AI. Upload a photo of your space, select from 30+ premium styles (Heritage, Modern, Industrial, Luxury), and customize every detail from chair count to lighting.

## Features

- **Shop Floor Planner**: Visualize layout zones (Chairs, Wash Stations, Lounge).
- **AI Design Wizard**: Vertical rail stepper for easy design workflow.
- **Moodboard Style Picker**: 4-tile collage style selection for premium feel.
- **Custom Controls**: Fine-tune specific elements like chair count, mirror type, and lighting.
- **Design Gallery**: Save, compare, and share your designs.
- **Premium Assets**: 30+ Styles, royalty-free assets.

## Tech Stack

- **Flutter**: UI Framework.
- **RevenueCat**: Subscription management.
- **Replicate**: AI Image Generation (Stable Diffusion).
- **SQLite**: Local history storage.
