# Camper Van Interior AI

Transform your van into a dream mobile home with AI.

## Features
- **Upload & Visualize**: Take a photo of your empty van and see it transformed.
- **30+ Styles**: From Scandinavian Minimal to Rugged Off-Grid.
- **Custom Controls**: Adjust layout, bed orientation, materials, and lighting.
- **Compare Mode**: Swipe to see before/after transformations.
- **Build Notes**: Get specs on potential layouts.

## Architecture
- **Framework**: Flutter
- **State Management**: Provider / Controller pattern
- **Theme**: Road-Luxe (Midnight Dark Mode)

## Getting Started
1. Run `flutter pub get`
2. Run `flutter run`
